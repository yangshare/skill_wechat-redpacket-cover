## 二、封面图详细规范

### 基础规格

- **格式**：PNG / JPG / JPEG
- **尺寸**：957 × 1278 像素（宽 × 高）
- **文件大小**：不超过 500KB
- **色彩空间**：建议使用 sRGB
- **宽高比**：约 3:4（竖版）

### 安全区域说明

封面图应用至拆红包页时：

```
┌─────────────────────┐
│      黑色区域       │  ← 顶部被遮挡，避免重要信息
├─────────────────────┤
│                     │
│    红色区域         │  ← 头像、昵称、祝福语区域
│    (中心偏上)       │     图案不宜过于复杂
│                     │
├─────────────────────┤
│                     │
│    安全区           │  ← 最佳设计区域
│                     │
└─────────────────────┘
```

### 设计原则

1. 避免在顶部黑色区域放置重要信息
2. 中心红色区域图案不宜过于复杂，避免影响文字识别
3. 建议使用装饰性图案，弱化品牌宣传
4. 不可出现二维码、联系方式、商品图等

## 三、封面挂件详细规范

### 基础规格

- **格式**：PNG（必须带透明通道）
- **尺寸**：1053 × 1746 像素
- **文件大小**：不超过 300KB
- **色彩空间**：建议使用 sRGB

### 可编辑区域

```
┌─────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │  ← 不可编辑区
│ ▓┌───────────────────┐▓ │
│ ▓│                   │▓ │
│ ▓│    可编辑区域     │▓ │  ← 设计元素必须在此区域内
│ ▓│                   │▓ │
│ ▓│                   │▓ │
│ ▓└───────────────────┘▓ │
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │  ← 不可编辑区
└─────────────────────────┘
```

### 设计要求

1. 挂件图案**必须**在可编辑区内设计
2. 超出可编辑区将导致上传失败
3. 挂件应是一个图案或图标，具备造型和装饰纹样
4. 不应是单一的线条或线框
5. 不建议完全填充，影响红包外形辨识度

---

## 四、气泡挂件详细规范

### 基础规格

- **格式**：PNG（必须带透明通道）
- **尺寸**：480 × 384 像素（宽 × 高）
- **文件大小**：不超过 300KB
- **色彩空间**：建议使用 sRGB

### 不可编辑区域

```
┌──────────────────────────┐
│                          │
│    ┌────────────┐        │
│    │  不可编辑  │        │  ← 挂件不能进入此区域
│    └────────────┘        │
│                          │
│        可设计区域        │  ← 安全设计区
│                          │
└──────────────────────────┘
```

### 设计要求

1. 挂件图案**不能**进入不可编辑区域
2. 超出限制将导致上传失败
3. 建议使用小巧精致的装饰元素

---

## 五、封面故事详细规范

### 图片规格

- **格式**：PNG / JPG / JPEG
- **尺寸**：750 × 1250 像素
- **文件大小**：不超过 300KB
- **数量**：最多 5 张，可混排

### 视频规格（本项目不支持）

- **格式**：MP4
- **编码**：H.264/AVC
- **宽度**：≥ 720 像素
- **宽高比**：3:5 ~ 16:9
- **时长**：≤ 15 秒
- **码率**：≤ 1600 kbit/s
- **文件大小**：≤ 10MB

### 故事描述

- **字数限制**：最多 40 个汉字
- **展示位置**：图片底部，带黑色渐变遮罩

### 裁剪说明

```
┌─────────────────┐
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│  ← 顶部被裁剪
│┌───────────────┐│
││               ││
││   安全展示区  ││  ← 重要信息放此区域
││               ││
│└───────────────┘│
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│  ← 底部被裁剪
└─────────────────┘
```

## 六、审核标准要点

### 禁止内容

1. **违法违规**：反对宪法、危害国家安全、泄露国家秘密等
2. **低俗内容**：淫秽、色情、赌博、暴力等
3. **商业广告**：二维码、联系方式、小程序码、条形码、口令等
4. **侵权内容**：未经授权的商标、肖像、版权作品等

### 设计原则

1. 封面必须**原创**或拥有**版权**
2. 风格美观简洁，封面图和挂件风格协调统一
3. 强调**装饰性**，弱化品牌和商品宣传
4. 避免过于商业化，影响用户使用意愿

### 肖像权

- 尽量避免出现真实人物肖像
- 如需使用，必须提供肖像权授权书
